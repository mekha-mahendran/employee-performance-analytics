Employee Performance Analytics

A full-stack analytics project using Java, MySQL, SQL, Power BI, HTML, CSS, and JavaScript

Java JDBC MySQL Power BI SQL HTML CSS JavaScript

About the Project

This project analyzes employee performance data using a Java application connected to a MySQL database via JDBC. It calculates department-wise salary and performance trends using SQL, and visualizes them through an interactive Power BI dashboard. This showcase website itself is built with HTML, CSS, and JavaScript, including an interactive sortable data table.

Key Insights

  • IT department has both the highest average salary and highest average performance rating.
  • Sales department shows the lowest performance and salary averages, suggesting a need for support or training.
  • HR department performs moderately across both metrics.

Tech Stack

  • Java (JDBC): Connects to MySQL and runs queries programmatically
  • MySQL: Stores employee records (id, name, department, salary, performance rating)
  • SQL: GROUP BY and AVG used for department-wise analysis
  • Power BI: Live dashboard connected directly to the MySQL database
  • HTML/CSS: Structures and styles this project showcase website
  • JavaScript: Powers the interactive sortable employee table above
